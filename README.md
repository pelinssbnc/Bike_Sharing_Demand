# 🚴 Bisiklet Kiralama Talep Tahmini

Hava koşulları, mevsimsel örüntüler ve zamana bağlı özellikler kullanılarak
saatlik bisiklet kiralama sayısını makine öğrenmesi ile tahmin eden proje.

---

## 📌 Genel Bakış

Bu proje **Kaggle Bike Sharing Demand** yarışmasını temel almaktadır.
Amaç; geçmiş kullanım örüntüleri, hava durumu ve mevsim verileriyle
saatlik bisiklet kiralama sayısını tahmin etmektir.

Temel hedefler:
- Zaman ve hava koşullarına göre talep örüntülerini keşfetmek
- Anlamlı özellikler türetmek (saat, haftanın günü, mevsim, hava)
- Regresyon modellerini karşılaştırarak RMSLE'yi minimize etmek
